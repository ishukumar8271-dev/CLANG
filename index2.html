Problem Statement

Manual identification of cattle and buffalo breeds is slow, prone to errors, and depends heavily on expert knowledge. This project proposes an automated, AI-powered image-based breed recognition system for better livestock management in India.

Problem Details

  • Ministry: Ministry of Fisheries, Animal Husbandry & Dairying
  • Problem Code: SIH25004
  • Category: Software
  • Domain: Agriculture, FoodTech & Rural Development

Proposed Solution

We will train a deep learning model (CNN) on labeled images of Indian cattle and buffalo breeds. The frontend allows users to upload an image, which is processed by a backend API (Flask/FastAPI) to return the predicted breed.
